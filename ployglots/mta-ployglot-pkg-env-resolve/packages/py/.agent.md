## Import Surface

Re-exported from `env_resolve/__init__.py` (alphabetized for stable diffs):

- `TRUTHY_STRINGS` â€” `tuple[str, ...] = ("true", "1", "yes", "on")`. Case-insensitive members are truthy under `resolve_bool`.
- `resolve(arg, env_keys, config, config_key, default)` â€” generic four-tier resolver.
- `resolve_bool(arg, env_keys, config, config_key, default)` â€” coerces to `bool`.
- `resolve_float(arg, env_keys, config, config_key, default)` â€” coerces to `float`; rejects partial-numeric strings (D6).
- `resolve_int(arg, env_keys, config, config_key, default)` â€” coerces to `int`; rejects decimal-shaped strings (D4) and `bool` inputs (D5).

## Canonical Usage

```python
from env_resolve import resolve, resolve_bool, resolve_int

# Tier 1 â€” explicit arg always wins.
region = resolve(
    cli_args.region,                  # arg
    ["REGION", "AWS_REGION"],         # env_keys (first-match wins)
    app_config,                       # config
    "region",                         # config_key
    "us-east-1",                      # default
)

# Coercion cascade: env-driven feature flag.
enabled = resolve_bool(None, "FEATURE_FLAG", None, None, False)

# Strict integer parsing â€” "3.14" returns the default, not 3.
port = resolve_int(None, "PORT", None, None, 8080)
```

## Parity Notes

- Surface is naming-convention difference only: `resolve_bool` â†” `resolveBool`, `env_keys` â†” `envKeys`, etc. Behavior is identical by spec.
- Python reads env via `os.getenv(k)`; TS via `process.env[k]`. Both return `str | None` / `string | undefined`.
- D5 (bool rejected by `resolve_int`) is the only place Python needs an explicit guard the TS port doesn't, because `bool` is a subclass of `int` in Python: `isinstance(val, bool)` MUST run *before* the `isinstance(val, int)` arm.
- D7 (`env_keys=None` short-circuit) is the only place the original Python source raised `TypeError`; the canonical spec tolerates nullish, so the port now early-returns to tier 3.
- See [`/.agents/parity.md`](../../.agents/parity.md) for the full drift catalog.
