# `polyglot-fetch-http-client` — Python skill

Async HTTP client for Python ≥ 3.11 built on [`httpx`](https://www.python-httpx.org/). Python twin of [`@polyglot/fetch-http-client`](../ts/.agent.md) — same surface, same defaults, same examples. `snake_case` where `httpx` is `snake_case`.

**Distribution name**: `polyglot-fetch-http-client`. **Import name**: `fetch_http_client`. Don't mix them up.

## Install

```bash
# Normal consumer install (after publish)
pip install polyglot-fetch-http-client

# Workspace-local, editable + dev extras (the day-to-day form)
make -C packages/py install
# or, from this directory:
uv pip install -e ".[dev]"
```

Runtime dep: `httpx >= 0.27`. Dev extras: `pytest`, `pytest-asyncio`, `pytest-cov`, `ruff`, `respx`.

The package is wired into a **uv workspace** — the root [`pyproject.toml`](../../pyproject.toml) declares `[tool.uv.workspace] members = ["packages/py"]` and `tool.uv.sources` pins it as `{ workspace = true }`. One `uv.lock` for the whole repo.

## Canonical usage pattern

```python
import asyncio
from fetch_http_client import AsyncClient, BasicAuth, Timeout

async def main():
    async with AsyncClient(
        base_url="https://api.example.com",
        auth=BasicAuth("user", "pass"),
        timeout=Timeout(connect=5.0, read=30.0),
    ) as client:
        resp = await client.get("/users/me")
        resp.raise_for_status()
        print(resp.json())

asyncio.run(main())
```

**Always use `async with`.** `AsyncClient.__aexit__` closes the underlying `httpx.AsyncClient`; without it, connections stay open until GC.

## Proxy convention

`Proxy` accepts a URL plus optional auth. Four idioms:

```python
from fetch_http_client import AsyncClient, Proxy, ProxyAuth

AsyncClient(proxy=Proxy(url="http://corp:3128"))
AsyncClient(proxy=Proxy(url="http://corp:3128", auth=ProxyAuth("svc", "secret")))
AsyncClient(proxy="http://corp:3128")       # string shorthand
AsyncClient(proxy={})                       # empty dict → auto-detect from HTTPS_PROXY / HTTP_PROXY
```

`NO_PROXY` (comma-separated hosts, wildcard `*`, suffix `.example.com`) honored via `should_bypass_proxy(host)`.

In examples, the convention is:

```python
from examples._shared import build_proxy
proxy = build_proxy({})                     # auto-detect default
kwargs = {"base_url": host, "auth": BasicAuth(u, p)}
if proxy is not None:
    kwargs["proxy"] = proxy                 # omit entirely when None
async with AsyncClient(**kwargs) as client:
    ...
```

## Retry + resilience

```python
from fetch_http_client import RetryConfig, JitterStrategy

retry = RetryConfig(
    max_attempts=5,
    base_delay=0.25,
    max_delay=30.0,
    multiplier=2.0,
    jitter=JitterStrategy.FULL,     # NONE | FULL | EQUAL | DECORRELATED
    retry_on_status={429, 500, 502, 503, 504},
)
client = AsyncClient(retry=retry)
```

Retries only fire for methods in `RetryConfig.retry_methods` — default is `IDEMPOTENT_METHODS` (`GET`, `HEAD`, `OPTIONS`, `TRACE`, `PUT`, `DELETE`). **`POST` / `PATCH` never retry.** Don't "fix" this — it's the same contract as the TS twin.

Circuit breaker:

```python
from fetch_http_client import CircuitBreaker, CircuitBreakerConfig

cb = CircuitBreaker(config=CircuitBreakerConfig(
    failure_threshold=5, recovery_timeout=30.0, success_threshold=2,
))
client = AsyncClient(circuit_breaker=cb)
```

States: `CLOSED` → `OPEN` (short-circuits with `CircuitOpenError`) → `HALF_OPEN` (probing window) → back to `CLOSED` or `OPEN`.

## Cache contract

Only `GET` / `HEAD` responses with status in `{200, 203, 300, 301, 404, 410}` are cached (RFC 7234-ish).

```python
from fetch_http_client import AsyncClient, CacheConfig, CachingClient

async with AsyncClient(base_url="https://x") as inner:
    async with CachingClient(inner, config=CacheConfig(ttl_seconds=60)) as client:
        await client.get("/thing")          # miss → set
        await client.get("/thing")          # hit
        print(client.cache.stats().as_dict())
```

`MemoryStorage` is the default backend; implement `CacheStorage` for alternatives.

Intentional drift:

- TS = `camelCase` (`raiseForStatus`), Python = `snake_case` (`raise_for_status`).
- Streaming primitives deferred — `Response` reads the full body today. TS has them.
- FastAPI adapter deferred — plan feature 13. (TS twin will ship a Fastify adapter; both land together.)
- No sync `Client` (neither twin has one; both are async-only).

## Gotchas

- **Distribution vs import name**: install `polyglot-fetch-http-client`, import `fetch_http_client`. The mismatch is intentional (PyPI namespace vs PEP 8 package name).
- **`async with` required** — exiting without it leaks `httpx.AsyncClient` connections.
- **100% coverage gate** — a new module with no tests will fail `make test` even if the logic works. Write the tests.
- **DeprecationWarning is an error** in tests (`filterwarnings = ["error::DeprecationWarning"]`). If `httpx` emits one, address it; don't silence.
- **`uv` workspace** — `uv sync` at the workspace root regenerates a single `uv.lock` for both the root and `packages/py`. Don't run `uv sync` inside `packages/py/` in isolation; it will desync.
- **Private modules (`_foo.py`)** — the barrel `__init__.py` is the contract; don't reach past it from consumer code.
