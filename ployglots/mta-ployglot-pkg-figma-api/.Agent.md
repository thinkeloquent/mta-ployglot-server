# Agent Integration Guide

How downstream coding agents (Claude Code, Cursor, Aider, …) should
consume this repo.

## What this repo delivers

Two twin packages that wrap the Figma REST API on top of
`@polyglot/fetch-http-client`:

- `packages/ts/` → `@polyglot/figma-api`  (Node ≥ 20, ESM `NodeNext`)
- `packages/py/` → `polyglot-figma-api`  (Python ≥ 3.11)

Each side ships:

- `FigmaClient` + all 8 domain sub-SDKs (`me`, `files`, `comments`,
  `projects`, `components`, `variables` (enterprise), `devResources`
  / `dev_resources`, `libraryAnalytics` / `library_analytics`
  (enterprise), `webhooks` (v2)).
- Figma-sensible retry defaults with a `forceOverwriteRetry` /
  `force_overwrite_retry` feature flag to opt out of the presets.
- Runnable examples across SDK / CLI / API / **servers** trees.
  Server examples cover three framework patterns each:
  Fastify (plugin / lifecycle / decorators) and FastAPI (lifespan /
  dependency-injection / middleware).
- Unit tests (57 TS, 51 py) + env-gated integration tests + CI
  Makefile.

## Env contract

Every example + integration test reads:

```
FIGMA_HOST       optional, default https://api.figma.com
FIGMA_USER       optional placeholder (Figma is token-only)
FIGMA_PASS       required — Figma Personal Access Token
HTTPS_PROXY      optional outbound proxy (auto-detected by proxy = {})
HTTP_PROXY_USER  optional proxy auth user
HTTP_PROXY_PASS  optional proxy auth pass
```

## Proxy contract

`new FigmaClient({ token, proxy: {} })` auto-detects from the
environment. `proxy: {}` is the **empty-placeholder** call — callers
opt in to env auto-detection by passing it literally. To override,
pass `{ host, user, pass }`. When no proxy is discoverable, the
helper returns `undefined` and the underlying fetch client is
constructed without a `proxy:` field.

See `packages/ts/src/proxy.ts` and `packages/py/figma_api/proxy.py`
for the helper.

## Usage

### Fresh run

```
make ci                      # install + lint + test + build in both packages
```

### Integration smoke

```
FIGMA_PASS=$YOUR_TOKEN make test-integration
```

Skips gracefully when `FIGMA_PASS` is unset.

### Per-package

```
make -C packages/ts ci       # TypeScript only
make -C packages/py ci       # Python only
```

## Conventions

- **Paths** are absolute or rooted at the workspace root.
- **Deferred work** (Fastify/FastAPI server, 4 of 8 domain clients,
  compose helper with custom retry defaults) is listed in each
  package's README under `## Deferred`. These are intentionally
  unimplemented and would ride a follow-up pass against the same
  plan tree at
  `../AI-Agent-Plans/figma-api-fetchclient-polyglot-20260423-7d4e9f1a/`.
- **Fetch client** is linked from the sibling workspace at
  `../mta-ployglot-pkg-fetch-client/` via `file:` (npm) and
  `tool.uv.sources` (uv). Rebuild it (`make -C ../mta-ployglot-pkg-fetch-client/packages/ts build`)
  after pulling changes there.
