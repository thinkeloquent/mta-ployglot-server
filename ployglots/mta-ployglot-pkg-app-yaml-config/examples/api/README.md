# API Examples

Public surface contract for `app-yaml-config`.

## Surface kind

Class-based + free-function exports. mjs uses ESM named exports; py uses module-level classes/functions.

| 03  | [Merge functions](03-merge-functions.md)                 | `deepMerge`, `mergeFiles`, `mergeGlobalIntoProviders`. |
