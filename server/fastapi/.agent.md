# mta-fastapi-server (py)

Runtime app. A standalone FastAPI server that boots on top of `fastapi_server.bootstrap` (the Python half of the `mta-ployglot-server-bootstrap` sibling, symlinked into `ployglots/`), with auto-discovered environment / lifecycle / route addons.

## Run

```bash
# from repo root
make dev-fastapi     # foreground, uvicorn
# or
make dev             # background both runtimes, logs under .dev/
```

Inside this package:

```bash
cd server/fastapi
python3 -m venv .venv
.venv/bin/pip install -e ../../ployglots/mta-ployglot-server-bootstrap/fastapi_server \
                     -e ../../../mta-ployglot-server-print-routes/packages/py \
                     'uvicorn[standard]>=0.30'
.venv/bin/python main.py   # uses env PORT (default 52000), HOST, APP_ENV, LOG_LEVEL
```

This package is `package-mode = false` (poetry). It is not published.

## Extension Contract

The bootstrap's route/lifecycle/environment addons auto-discover files under `config/` by filename suffix — symmetric to the Fastify twin. Drop a file with the matching suffix and it is loaded at boot. Numeric prefix sorts loading order.

| Directory | Suffix | Export shape | Discovery order |
|---|---|---|---|
| `config/environment/` | `.env.py` | side-effect module (mutates `os.environ`) | ascending by numeric prefix |
| `config/lifecycles/` | `.lifecycle.py` | `def on_init(...)?`, `async def on_startup(...)?`, `async def on_shutdown(...)?` | ascending by numeric prefix |
| `config/routes/` | `.routes.py` | `def mount(app, config) -> APIRouter` | ascending by numeric prefix |

Adding a capability to this server means adding a file in one of those three directories — no edits to `main.py` or the bootstrap sibling required.

## Canonical Usage

A new route file:

```python
# config/routes/20_hello.routes.py
from fastapi import APIRouter, FastAPI


def mount(app: FastAPI, config) -> APIRouter:
    router = APIRouter()

    @router.get("/hello")
    async def hello():
        return {
            "service": config.title,
            "profile": config.profile,
            "hello": "world",
        }

    return router
```

A new lifecycle hook:

```python
# config/lifecycles/15_db_pool.lifecycle.py
import os

async def on_init(app, config):
    app.state.db_pool = await create_pool(os.environ["DB_URL"])

async def on_shutdown(app, config):
    await app.state.db_pool.aclose()
```

After saving either file, `make dev-fastapi` picks it up on next boot (no explicit registration).

## Integration

- **Depends on (monorepo):** `thinkeloquent-fastapi-server` via `path = "../../ployglots/mta-ployglot-server-bootstrap/fastapi_server"` with `develop = true` (sibling-repo symlink under `ployglots/`), `print-routes-fastapi` via sibling repo path.
- **Depends on (external, transitively):** `fastapi`, `starlette`, `uvicorn[standard]`.
- **Environment read:**
  - `PORT` (default `52000`), `HOST` (default `0.0.0.0`), `APP_ENV` (default `local`), `LOG_LEVEL` (default `info`)
  - `BUILD_ID`, `BUILD_VERSION` — surfaced in `/healthz` payload
  - `LOG_DIR` — when set, appends per-request JSON to `<LOG_DIR>/fastapi.request.log` and errors to `fastapi.error.log`
  - `PYTHON_ENV` — honored by downstream fetch-client consumers (see bootstrap)

## Parity Notes

Intentional drift with `mta-fastify-server`:

- **Suffix:** `.py` here, `.mjs` there.
- **Export shape:** `mount(app, config) -> APIRouter` (sync factory). FastAPI side can't await inside `register_routes`; if async init is needed, stash a coroutine on `app.state` and run it from an `on_startup` hook.
- **Lifecycle timing:** `on_init` is sync, runs during `setup()`. `on_startup` / `on_shutdown` attach to FastAPI's `lifespan` context manager.
- **Request state:** ASGI middleware clones `initial_state` into `request.state.state` per-request (vs. Fastify's `decorateRequest` + `onRequest` hook).
- **Default port:** `52000` vs. `51000`.
- **Install:** bypasses poetry — the root `Makefile`'s `install-python` target runs `pip install -e` directly against the sibling-symlinked paths under `ployglots/` (see `Makefile` line 80 region).
